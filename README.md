# Interactive Story for Children

An interactive story inspired by classic "choose your own adventure" books, created as my very first JavaScript project.

## Why I built it

I wanted to create a story that could one day encourage my son to enjoy reading while also giving myself a flexible playground to learn front-end development from scratch.

As I learned new concepts, I kept expanding the project, turning it into a personal experimentation lab for JavaScript, HTML and CSS.

## Tech Stack

- HTML
- CSS
- Vanilla JavaScript

## Status

Personal project.
The story is largely complete and the repository represents my first exploration of front-end development.
